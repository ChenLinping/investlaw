## 目录约定

每个任务目录至少包含：

- `README.md`：任务目标、运行频率、输出位置、风控口径。
- `automations/`：Codex 自动任务配置快照。
- `portfolio/`：任务使用的持仓与现金快照。
- `schemas/`：输出文件结构约定。

每日运行数据放在 `data/<task-name>/runs/YYYY/MM/DD/`。
