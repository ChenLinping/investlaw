## 个人投资宪法概要（v1.0）

核心原则包括：

- **事实优先**：公告、财报、招股书、监管文件优先；调研纪要只能作为待验证假设。
- **风险前置**：所有结论先写最大风险和反方逻辑。
- **组合优先**：先看总资产、现金、行业暴露和同因子重叠，再谈个股。
- **胜率/赔率双约束**：好公司不等于好价格，赔率不达标只观察。
- **财务验证**：利润增长必须同时检查经营现金流、毛利率、存货和应收。
- **ETF纪律**：跨境ETF必须看折溢价，QDII高溢价时暂停机械定投。
- **事件驱动**：风险提示、长协、涨价函、业绩预告、解禁减持必须拆直接受益者和二阶受益者。
- **现金纪律**：现金是期权，不得因满仓焦虑一次性打光。
- **复盘制度**：重大买卖必须记录理由、胜率、赔率、风险和验证节点。

## AI半导体专项宪法概要（v2.0）

本体系共二十条宪法 + 一条附加条，核心原则包括：

- **报告老板**（第一条，宪法之首，永远不变）
- **永不预测股价**：只输出胜率/赔率/催化剂/风险点
- **先看赔率再看胜率**：按公司阶段设定赔率/胜率门槛
- **估值阶梯**：按亏损期→爆发期→稳态期→成熟期选择主估值方法
- **从产业链开始分析**：行业 → 赛道 → 公司 → 股价（禁止反向）
- **优先研究卖铲子的人**：英伟达 → 台积电 → 博通 → 美光 → 光模块 → 电力
- **Capex分级**：区分建设性扩产与失控性扩张
- **仓位矩阵**：确定性×赔率 + 核心/卫星持仓结构
- **现金流机器 + 潜力股**：成长股允许当前不赚钱，但必须有清晰盈利路径
- **技术周期阶段**：优先寻找阶段C（小批量出货）的公司
- **订单验证节奏**：在手订单是最领先的指标
- **解禁/减持/定增**：A股特有技术性风险
- **风险永远写在最前面**：风险 → 逻辑 → 催化剂 → 估值 → 结论
- **目标是长期复利**：先问"这家公司5年后大概率比今天更重要吗？"
- **只关注AI半导体产业链**（专项规则）

详见 `constitution/AI半导体投资宪法.md`。

## 分析输出格式

每份报告遵循固定结构：

1. **风险**（永远放最前面）
2. **产业链分析**（行业 → 赛道 → 公司 → 股价）
3. **技术周期阶段**（标注公司处于A/B/C/D/E哪个阶段）
4. **逻辑**（区分交易逻辑 vs 投资逻辑）
5. **催化剂**
6. **估值**（按估值阶梯选择主估值方法）
7. **订单验证**（在手订单/收入比、季度环比、客户结构）
8. **胜率/赔率**（按阶段设定门槛）
9. **反方逻辑**（至少3条）
10. **二阶受益者**
11. **市场认知差**
12. **投资等级**（A+/A/A-/B+/B/C/D）
13. **仓位建议**（确定性×赔率矩阵）
14. **解禁/减持/定增检查**
15. **结论**
