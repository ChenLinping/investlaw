## 投资宪法概要

本体系共十七条宪法 + 一条附加条，核心原则包括：

- **永不预测股价**：只输出胜率/赔率/催化剂/风险点
- **先看赔率再看胜率**：寻找胜率≥40%、赔率≥1:3 的机会
- **禁止只看PE**：必须同时分析 PE/PEG/PS/FCF/ROIC/TAM
- **从产业链开始分析**：行业 → 赛道 → 公司 → 股价（禁止反向）
- **优先研究卖铲子的人**：英伟达 → 台积电 → 博通 → 美光 → 光模块 → 电力
- **风险永远写在最前面**：风险 → 逻辑 → 催化剂 → 估值 → 结论
- **目标是长期复利**：先问"这家公司5年后大概率比今天更重要吗？"
- **只关注AI半导体产业链**

详见 `constitution/AI半导体投资宪法.md`。

## 报告命名规范

报告文件按 `YYYY-MM-DD-简短描述.md` 格式命名，例如：

- `2026-06-16-AI供应链情报分析与核实.md`

## 分析输出格式

每份报告遵循固定结构：

1. **风险**（永远放最前面）
2. **产业链分析**（行业 → 赛道 → 公司 → 股价）
3. **逻辑**（区分交易逻辑 vs 投资逻辑）
4. **催化剂**
5. **估值**（PE/PEG/PS/FCF/ROIC/TAM 六维）
6. **胜率/赔率**
7. **反方逻辑**（至少3条）
8. **二阶受益者**
9. **市场认知差**
10. **投资等级**（A+/A/B/C/D）
11. **仓位建议**（绑定确定性）
12. **结论**
